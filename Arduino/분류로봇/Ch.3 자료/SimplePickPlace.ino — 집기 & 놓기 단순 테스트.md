# 🦷 SimplePickPlace.ino — 집기 & 놓기 단순 테스트

## 📌 한 번에 확인하는 것

> **15번 저장 전에 이 코드를 업로드하면 안 돼.**
> 15, 16, 17, 18번 자세가 저장된 상태에서 실행하는 코드야.

---

## 🔄 동작 순서

```
15번 (픽업 준비)
    ↓
16번 (블록에 닿는 위치)
    ↓
그리퍼 닫기 ← 블록 집기
    ↓
15번으로 복귀 ← 들어올리기
    ↓
17번 (분류대 이동 준비)
    ↓
18번 (분류대 앞 배치 위치)
    ↓
그리퍼 열기 ← 블록 내려놓기
    ↓
17번으로 복귀
```

---

## 🚀 사용 방법

### 1단계 — 업로드
`SimplePickPlace.ino` 업로드

### 2단계 — 시리얼 모니터
```
115200 baud / Newline(\n)
```

### 3단계 — 명령어

| 명령어 | 동작 |
|--------|------|
| `go` | **전체 동작** 1회 (집기 + 놓기) |
| `pick` | 집기만 (15→16→닫기→15) |
| `place` | 놓기만 (17→18→열기→17) |
| `open` | 그리퍼 열기 |
| `close` | 그리퍼 닫기 |
| `home` | 15번 준비 자세로 이동 |
| `help` | 가이드 출력 |

## 🎬 시나리오 — 단계별 테스트

### 맨 처음 테스트

**집기만 먼저:**
```
[입력] pick

[출력]
=== 집기 시작 ===
[1] 픽업 준비 자세(15번)...
[2] 픽업 자세(16번) — 블록 위치로 내려감...
[3] 그리퍼 닫기 — 블록 집기
[4] 픽업 준비 자세(15번)로 복귀 — 들어올리기...
=== 집기 완료 ===

→ 블록이 제대로 집혀졌는지 눈으로 확인!
```

**놓기만 테스트:**
```
[입력] place

[출력]
=== 놓기 시작 ===
[5] 분류대 이동 준비(17번)...
[6] 분류대 앞 배치 자세(18번)...
[7] 그리퍼 열기 — 블록 내려놓기
[8] 분류대 이동 준비(17번)로 복귀...
=== 놓기 완료 ===
```

**전체 동작:**
```
[입력] go

[출력]
[GO] 전체 동작 시작
... (집기 동작) ...
... (놓기 동작) ...
[GO] 전체 동작 완료!
🟢 초록 LED
```

## ⚠️ 조정이 필요할 때

### 팔이 너무 빨리 내려가는 경우

```cpp
// 코드 상단 #define에서 수정
#define TIME_READY   1200  // 높이기
#define TIME_PICKUP  1500  // 늘리기 ← 이 값을 늘려
#define TIME_PLACE   1200
```

### 집게 위치가 빗나가는 경우

```
manageManipulatorPose.ino로 교체
  off
  save → 16,PICKUP   (다시 저장)
  on → run → 16,1500 (확인)
```

### 자세 번호 바꾸고 싶으면

```cpp
// 코드 상단 #define에서
#define POSE_PICKUP_READY   15   // 다른 번호로 변경 가능
#define POSE_PICKUP         16
#define POSE_PLACE_READY    17
#define POSE_PLACE          18
```

## ✅ 확인 체크

- [ ] `pick` 명령으로 블록 제대로 집히는지 확인
- [ ] `place` 명령으로 정확한 위치에 내려놓는지 확인
- [ ] `go` 명령으로 전체 동작 3회 연속 반복 확인
- [ ] 판에서 블록이 떨어지지 않는지 확인
