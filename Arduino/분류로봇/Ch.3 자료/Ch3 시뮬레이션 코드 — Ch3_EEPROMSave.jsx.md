# 💻 Ch3 시뮬레이션 코드 — Ch3_EEPROMSave.jsx

## 📌 실행 방법

1. 아래 코드를 `Ch3_EEPROMSave.jsx` 파일로 저장
2. VSCode Live Server로 실행
3. 3개 탭 활용:
	- **📋 절차 안내** — 저장 7단계 클릭형 가이드
	- **💾 저장 시뮬** — 자세 선택 → 저장 → 실행 확인
	- **🗂️ EEPROM 맵** — 메모리 주소 구조 확인

## 📄 7단계 저장 절차

| # | 단계 | 설명 |
|---|------|------|
| 1 | 💻 저장용 코드 업로드 | SavePose.ino를 Arduino IDE에서 업로드 |
| 2 | 📺 시리얼 모니터 열기 | 9600bps, 줄끝: Newline 설정 필수! |
| 3 | ⬇️ off → 토크 OFF | 토크를 꺼야 손으로 팔을 움직일 수 있음 |
| 4 | ✋ 팔을 원하는 위치로 이동 | 한 손으로 받치면서 천천히 위치 잡기 |
| 5 | 💾 save → 저장 모드 | save 입력 후 엔터 |
| 6 | ⌨️ 번호,이름 입력 | 예: `1,INITIAL` → 초록불 = 저장 성공 |
| 7 | ▶️ run으로 동작 확인 | on → run → `1,1000` 입력으로 재생 확인 |

## 🗂️ EEPROM 주소 매핑

```cpp
EEPROM.put(40 * (id - 1), pose); // 저장
EEPROM.get(40 * (id - 1), pose); // 읽기
// 전원 꺼도 유지!
```
