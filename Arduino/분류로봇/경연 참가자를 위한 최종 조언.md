# 🗓️ D-1 (경연 전날)

## 반드시 완료해야 할 것

- [ ] `list` 명령어로 EEPROM 자세 11개 전부 저장 확인
- [ ] `run`으로 모든 자세 실제 동작 눈으로 확인
- [ ] `backup` 명령어로 EEPROM 백업 텍스트 파일로 저장
- [ ] 대회 코드 업로드 완료 상태로 마무리
- [ ] 배터리 완충

## 절대 하지 말 것

- ❌ 전날 밤 코드 대규모 수정 — 검증 시간 없음
- ❌ EEPROM clear 후 재저장 없이 잠들기
- ❌ 배터리 방전 상태로 두기

# 🌅 D-Day 아침 (경연 1~2시간 전)

## 1. 현장 도착 즉시 — 조명 확인

Pixy2 카메라는 **조명에 가장 민감**해.

```
연습실 조명 ≠ 경연장 조명
→ 형광등 색온도, 밝기가 다르면 sig 값이 달라짐
→ 현장 조명 아래에서 반드시 재학습!
```

순서:
1. 경연장 조명 켜진 상태에서 PixyMon 연결
2. 블록 4개 sig 재학습 (밝은 면만 드래그)
3. Range 8.0부터 조정
4. 4색 모두 교차 오염 없이 감지되는지 확인

## 2. PSD 실측

```
Module_Ch4_PSDTest.ino 업로드
→ 로봇을 슬롯1 앞에 수동 위치
→ 시리얼 모니터에서 SL 값 확인
→ SLOT_PSD[0] 수정
→ 슬롯 2,3,4 반복
→ 대회 코드 재업로드
```

> ⚠️ 이 작업만 빠뜨려도 블록을 못 찾아. 절대 생략 금지.

## 3. 전체 1회 완주 테스트

실제 블록 놓고 SW1 눌러서 처음부터 끝까지 한 번 돌려봐.

- 블록 4개 인식되는지
- 슬롯 이동 정확한지
- 집기/배치 자세 정확한지
- 복귀 정상인지

---

# 🎯 경연 직전 5분

```
1. 로봇을 출발 위치에 정확히 세우기
2. 블록 4개 경연 규정대로 배치 확인
3. 배터리 전압 확인 (11V 이상)
4. 대회 코드 업로드 상태인지 재확인
5. SW1 버튼 눌러서 시작
```

# 🚨 현장 돌발상황 대응

## 상황별 즉각 대응

| 증상 | 원인 | 대응 |
|------|------|------|
| 팔이 안 움직임 | EEPROM 누락 | backup 텍스트로 restore |
| 블록 못 찾음 | 카메라 sig 오류 | PixyMon 재학습 |
| 슬롯 엉뚱한 곳 서 | SLOT_PSD 미보정 | PSDTest 코드로 실측 |
| 집다가 놓침 | GRIP_PICK 자세 오류 | manageManipulatorPose run으로 재확인 |
| 색깔 오분류 | Range 값 겹침 | Range 낮추거나 재학습 |
| 팔이 처짐 | 중력 + P게인 부족 | operatingTime 늘리기 |

## 시간 없을 때 우선순위

```
1순위: 카메라 재학습 (5분)
2순위: SLOT_PSD 실측 (5분)
3순위: run으로 자세 확인 (3분)
```

# 💡 멘탈 관리

## 테스트 중 안 되는 게 있을 때

> **"한 번에 하나씩만 고친다."**

두 개 동시에 고치면 뭐가 원인인지 모른다. 반드시 하나 바꾸고 → 테스트 → 확인 → 다음.

## 경연 중 실수했을 때

- 당황하지 말고 리셋 후 재시도
- 부분 점수가 있으면 할 수 있는 것만 해도 점수
- 멈춘 로봇 앞에 서 있지 말고 審判에게 상황 알리기
