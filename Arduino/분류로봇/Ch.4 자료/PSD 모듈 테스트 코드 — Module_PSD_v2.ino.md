# 🧪 PSD 모듈 테스트 코드 — Module_PSD_v2.ino

## 📌 이 코드로 할 수 있는 것

파일 하나로 **5개 섹션**을 명령어로 선택 실행.
PSD 관련 모든 함수를 단독으로 테스트할 수 있어.

> ⚠️ **사전 조건:** EEPROM 자세 저장 완료 + SLOT_PSD 실측 완료 상태에서 실행

---

## ⚙️ 시리얼 모니터 설정

| 항목 | 값 |
|------|-----|
| 속도 | **9600 baud** |
| 줄끝 | **Newline (n)** |

---

## 🗺️ 섹션 구조

| 명령어 | 섹션 | 사용 함수 | 학습 포인트 |
|--------|------|-----------|-------------|
| `A` | 실시간 PSD 값 읽기 | `FL/FR/SL/SL430_PSD_Save()` | 각 센서 방향과 이동평균 필터 효과 |
| `B` | 전방 PSD 접근 | `PID_AlignAndApproach()` | cm 기준 PID 전진/후진 |
| `C` | 측면 PSD 좌우 이동 | `DriveLeftWithSideLeftSensorPID()` | 옆벽 기준 슬롯 이동 |
| `D` | 2축 동시 정밀 정렬 | `AlignUsingFrontAndSide430PID()` | FL + SL430 동시 직각 정렬 |
| `E` | 경계 감지 복귀 | `SL_PSD_Save()` 루프 | 15cm 기준 경계 3단계 통과 |

---

## 📋 섹션별 명령어 상세

### 섹션 A — 실시간 PSD 값 읽기

```
A
```

→ 200ms마다 4개 센서 값 출력. SW1 버튼 누르면 종료.
```
FL=25.3  FR=24.8  SL=18.2  SL430=6.1
FL=25.1  FR=24.9  SL=18.0  SL430=6.0
...
```

> 💡 손을 센서에 가까이/멀리 대면서 값 변화 확인

---

### 섹션 B — 전방 PSD 접근 (PID)

```
B        ← 명령어 목록 출력
b1       ← OBS(7.5cm) 접근
b2       ← OBSB(11.5cm) 후진 정렬
b3,20    ← 직접 20cm 입력
```

```cpp
// 내부 동작
PID_AlignAndApproach(dxl, OBS, 10, 0.01, 0.2);
//                         ↑       ↑
//                       목표cm  허용오차
```

---

### 섹션 C — 측면 PSD 좌우 이동

```
C        ← 명령어 목록 출력
c1       ← 슬롯1 (17.5cm)
c2       ← 슬롯2 (24.5cm)
c3       ← 슬롯3 (31.0cm)
c4       ← 슬롯4 (35.5cm)
csp      ← 기준점 SP (6cm)
cmdz     ← 미션존 MDZ (14cm)
c,25     ← 직접 25cm 입력
```

> ⚠️ 기준은 **적재함이 아니라 경연장 왼쪽 옆벽!**

---

### 섹션 D — 2축 동시 정밀 정렬

```
D        ← 기본 설정(FP=43, SP=6) 실행
d,40,5   ← 전방 40cm, 옆면 5cm 직접 입력
```

```cpp
// 내부 동작
AlignUsingFrontAndSide430PID(dxl, 43, 6,
  0.5, 0.5,          // 가중치
  5, 0.01, 0.1,      // 전방 PID
  2, 0.01, 0.03,     // 옆면 PID
  50, 150);          // 허용오차, 타임아웃
```

> 💡 이 정렬이 완료된 후에 슬롯 이동을 해야 SL값이 정확해!

---

### 섹션 E — SL PSD 경계 감지 복귀

```
E        ← 경계 감지 복귀 실행
etest    ← SL값 모니터링만 (SW1 종료)
```

```
15cm 기준:
  dis > 15  →  벽 없음 (빈 공간)
  dis < 15  →  벽 있음 (구조물 감지)

3단계 통과:
  1단계: dis > 15  (미션수행존 이탈)
  2단계: dis < 15  (미션지시존 감지)
  3단계: dis > 15  (미션지시존 벽 통과)
```

---

## 🎯 권장 테스트 순서

```
1. A → 센서 방향 확인 (손으로 각 센서 가리기)

2. B → 전방 접근 테스트
   b1 → 장애물 앞에 세우기
   b2 → 뒤로 빠지기

3. D → 적재함 앞 정렬 먼저!
   (정렬 안 하면 C의 SL값이 부정확)

4. C → 슬롯 이동 테스트
   c1 → c2 → c3 → c4 → csp 순서

5. E → etest로 15cm 경계 확인 먼저
   → E로 실제 복귀 실행
```

## 💡 코드 수정 위치

```cpp
// 거리 상수 (현장 실측 후 수정)
#define OBS   7.5
#define OBSB  11.5
#define MDZ   14.0
#define FP    43.0
#define SP    6.0

// 슬롯 거리 (당일 실측 필수!)
float SLOT_PSD[4] = {17.5, 24.5, 31.0, 35.5};
```
