# 🤖 색깔 블록 분류 로봇 — 경연 핵심 강의

이 문서는 색깔 블록 분류 로봇 경연을 준비하는 학생들을 위한 **핵심 강의 자료**입니다.

> 💡 역기구학·정기구학은 몰라도 됩니다. 이 문서에서 다루는 것만 완벽히 이해하면 경연은 돌아갑니다.

---

## 📚 챕터 구성

| 챕터 | 제목 | 핵심 내용 |
|------|------|-----------|
| Ch.1 | 전체 미션 구조 이해 | 로봇이 무엇을 하는지 |
| Ch.2 | Pixy2 카메라 세팅 | 블록 색깔 인식 학습 |
| Ch.3 | EEPROM 자세 저장 | 로봇팔 위치 가르치기 |
| Ch.4 | PSD 거리값 보정 | 슬롯 이동 거리 맞추기 |
| Ch.5 | 핵심 변수 연결 구조 | 코드가 어떻게 연결되는지 |
| Ch.6 | 당일 체크리스트 | 경연 당일 순서 |

---

## ⚠️ JSX 시뮬레이션 안내

각 챕터에 **JSX 시뮬레이션 파일**이 제공됩니다.
VSCode에서 Live Server로 열어서 함께 실습하세요.

```
실행 방법:
1. VSCode에서 해당 .jsx 파일 열기
2. Live Server 확장 설치 후 실행
3. 브라우저에서 시뮬레이션 확인
```

---

## 🏁 한 줄 요약

> **카메라로 순서 파악 → 팔 자세 저장 → 거리값 보정**
> 이 세 가지가 완벽하면 경연은 돌아간다.
