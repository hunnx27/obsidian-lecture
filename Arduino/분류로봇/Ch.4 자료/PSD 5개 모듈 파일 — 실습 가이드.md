# 📡 PSD 5개 모듈 파일 — 실습 가이드

> 파일 하나씩 업로드해서 그 동작만 테스트. 모든 파일 공통: **9600 baud / Newline(n)**

---

## 권장 실습 순서

```
A → 센서 방향 파악
B → 전방 접근 이해
D → 정렬 먼저!
C → 슬롯 이동 (D 정렬 후 해야 정확)
E → 경계 감지 복귀
```

---

## A — PSD 값 실시간 모니터링
**파일:** `A_PSD_Monitor.ino`
**학습 목표:** 4개 센서가 어느 방향을 바라보는지 직접 확인

```
업로드 → 시리얼 모니터 열기
→ 손을 각 센서 앞에 가져다 대기
→ 어느 값이 변하는지 확인
→ SW1 버튼으로 종료
```

```
FL=25.3  FR=24.8  SL=18.2  SL430=6.1
        ↑전방좌  ↑전방우  ↑좌측면  ↑좌측근거리
```

---

## B — 전방 PSD 접근
**파일:** `B_Front_Approach.ino`
**학습 목표:** FL PSD 기준으로 목표 cm까지 PID 전진/후진

```
명령어:
  1       → OBS(7.5cm) 접근
  2       → OBSB(11.5cm) 후진 정렬
  3,20    → 직접 20cm 지정
```

```cpp
// 내부 함수
PID_AlignAndApproach(dxl, OBS, 10, 0.01, 0.2);
//                         ↑cm  ↑허용오차
```

---

## C — 측면 PSD 좌우 이동
**파일:** `C_Side_Move.ino`
**학습 목표:** SL PSD 기준으로 슬롯 위치 이동 (기준은 경연장 왼쪽 옆벽!)

```
명령어:
  1     → 슬롯1 (17.5cm)
  2     → 슬롯2 (24.5cm)
  3     → 슬롯3 (31.0cm)
  4     → 슬롯4 (35.5cm)
  sp    → 기준점 (6cm)
  mz    → 미션존 (14cm)
  ,25   → 직접 25cm 지정
```

> ⚠️ **D(정렬) 완료 후 실행해야 SL값이 정확!**

---

## D — 2축 동시 정밀 정렬
**파일:** `D_2Axis_Align.ino`
**학습 목표:** FL + SL430 두 센서로 적재함에 직각 정렬

```
명령어:
  1         → 기본 정렬 (FP=43, SP=6)
  2,40,5    → 전방40cm, 측면5cm 직접 지정
  p         → 현재 FL, SL430 값 확인
```

```cpp
// 내부 함수
AlignUsingFrontAndSide430PID(dxl, 43, 6,
  0.5, 0.5,        // 전방:측면 가중치
  5, 0.01, 0.1,    // 전방 PID
  2, 0.01, 0.03,   // 측면 PID
  50, 150);        // 허용오차, 타임아웃
```

---

## E — 경계 감지 복귀
**파일:** `E_Boundary_Return.ino`
**학습 목표:** SL 15cm 기준으로 3단계 경계 통과 후 복귀

```
명령어:
  m   → SL값 모니터링 (15cm 기준 확인)
  r   → 경계 감지 복귀 실행
  p   → 현재 SL 1회 출력
```

```
15cm 기준:
  dis > 15 → 벽 없음 (빈 공간)
  dis < 15 → 벽 있음 (구조물)

3단계:
  1단계: dis > 15 (수행존 이탈)
  2단계: dis < 15 (지시존 감지)
  3단계: dis > 15 (지시존 벽 통과)
```
